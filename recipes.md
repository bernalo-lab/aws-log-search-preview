# Recipes

A collection of practical CloudWatch Logs Insights queries designed to solve real production debugging problems. Each recipe provides a ready-to-use search pattern along with operational context and guidance.

---


## Recipes

Each page in this section is a **stand-alone log search recipe**.

Every recipe includes:

- The problem it solves
- A CloudWatch Logs Insights query
- Why the query works
- When it can mislead
- Variations for different system shapes

You don’t need to read these in order.

Open the one that matches your situation, adapt it to your environment, and move on.
